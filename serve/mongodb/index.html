Mongodb

Install

Env

CentOS 7.4

Install mongodb

1、Check mongodb version

https://www.mongodb.com/download-center/community

  cd /usr/local/src

  wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-4.0.9.tgz

  tar -zxvf mongodb-linux-x86_64-4.0.9.tgz

  mv mongodb-linux-x86_64-4.0.9

  /usr/local/mongodb

2、Set up config

  cd /

  mkdir -p data/mongodb/db

  mkdir -p data/mongodb/logs

  cd data/mongodb

  vi mongodb.conf

3、Write this config in mongodb.conf

  dbpath = /data/mongodb/data
  logpath = /data/mongodb/logs/mongodb.log
  bind_ip = 0.0.0.0
  port = 2728
  fork = true

4、Save & Start mongodb

  cd /usr/local/mongodb/bin

  ./mongod –config /data/mongodb/mongodb.conf

5、Check mongodb

ps aux |grep mongodb

others

kill mongodb

kill -9 PID
