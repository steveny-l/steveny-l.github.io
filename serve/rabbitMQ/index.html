RabbitMQ

Install

Env

CentOS 8.0

1、Install socat

  yum install -y socat

2、Find out RabbitMQ Erlang version requirements and download rpm package

https://www.rabbitmq.com/which-erlang.html

download erlang-22.2.6-1.el8.x86_64.rpm Erlang download

download rabbitmq-server-3.8.2-1.el8.noarch.rpm RabbitMQ download

3、Install Erlang and RabbitMQ

  rpm -ivh erlang-22.2.6-1.el8.x86_64.rpm
  rpm -ivh rabbitmq-server-3.8.2-1.el8.noarch.rpm
